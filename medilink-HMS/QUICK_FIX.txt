                              CHOOSE YOUR IDE
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ VS CODE                                                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Install: "Extension Pack for Java" (by Microsoft)                       │
│ 2. Install: "Lombok Annotations Support for VS Code"                       │
│ 3. Press: Ctrl+Shift+P → "Reload Window"                                   │
│ 4. Press: Ctrl+Shift+P → "Java: Clean Java Language Server Workspace"     │
│ 5. Press: Ctrl+Shift+P → "Java: Build Project"                            │
│                                                                             │
│ Result: Red errors disappear ✅                                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ INTELLIJ IDEA                                                               │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. File → Settings → Plugins                                               │
│ 2. Search "Lombok" → Install → Restart                                     │
│ 3. File → Settings → Compiler → Annotation Processors                      │
│ 4. Check "Enable annotation processing" → Apply → OK                       │
│ 5. Right-click pom.xml → Maven → Reload Project                            │
│ 6. Build → Rebuild Project                                                 │
│                                                                             │
│ Result: Red errors disappear ✅                                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│ ECLIPSE                                                                     │
├─────────────────────────────────────────────────────────────────────────────┤
│ 1. Download: https://projectlombok.org/download                            │
│ 2. Run: java -jar lombok.jar                                               │
│ 3. Select Eclipse → Install → Restart                                      │
│ 4. Right-click project → Properties                                        │
│ 5. Java Compiler → Annotation Processing                                   │
│ 6. Check "Enable annotation processing" → Apply and Close                  │
│ 7. Project → Clean                                                         │
│                                                                             │
│ Result: Red errors disappear ✅                                            │
└─────────────────────────────────────────────────────────────────────────────┘
